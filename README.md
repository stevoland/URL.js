# URL.js

Manipluate URI parts in browser based Javascript

## Integration

### AMD
```javascript
// AMD
require(['path/to/URL.js'], function (URL) {
  // URL.parse('http://...');
});
```

### Vanilla JS
```html
<!-- Vanilla javascript -->
<script src="path/to/URL.js"></script>
<script>
	console.log(URL.parse('http://...');
</script>
```
